## Описание программы (кратко)
Movie Library — десктопное приложение на Python с графическим интерфейсом (Tkinter) для учёта и фильтрации коллекции фильмов. Программа позволяет добавлять фильмы с указанием названия, жанра, года выпуска и рейтинга, сохранять данные в JSON‑файл и фильтровать коллекцию по жанру и году.


### Возможности (что программа умеет делать)
* **Добавление фильмов** в библиотеку с заполнением следующих полей:
    * название;
    * жанр;
    * год выпуска (целое число);
    * рейтинг (число от 0 до 10).
* **Валидация данных** при вводе:
    * проверка заполнения всех полей;
    * контроль корректности года (целое число);
    * проверка диапазона рейтинга (0–10).
* **Сохранение коллекции** между запусками программы в файле `movies.json`.
* **Отображение фильмов** в виде интерактивной таблицы с колонками: «Название», «Жанр», «Год», «Рейтинг».
* **Фильтрация коллекции**:
    * по жанру (частичное совпадение, без учёта регистра);
    * по году выпуска (точное совпадение).
* **Визуальная обратная связь** через всплывающие сообщения об ошибках или успехе операций.
* **Очистка полей ввода** после добавления фильма.
* **Автоматическое обновление таблицы** при добавлении или фильтрации данных.

## Инструкция по запуску

### Требования для запуска программы
* Python 3.6 или выше.
* Стандартные библиотеки Python:
    * `tkinter` (GUI‑фреймворк);
    * `ttk` (расширение для tkinter);
    * `json` (работа с JSON‑файлами);
    * `os` (проверка существования файла).

Программа не требует установки дополнительных пакетов — все зависимости входят в стандартную поставку Python.

### Запуск (инструкция по запуску)
1. Сохраните код программы в файл с расширением `.py` (например, `movie_library.py`).
2. Убедитесь, что у вас установлен Python 3.6+.
3. Откройте терминал или командную строку.
4. Перейдите в директорию, где сохранён файл программы.
5. Запустите программу командой:
```bash
